{%- comment -%} Wspolny komponent nawigacji dla podstron Jekyll (post, lista wpisow, strona specjalizacji oraz — po dodaniu front-mattera — index2.html). Strona glowna (legacy index.html bez front-mattera) ma swoja kopie nawigacji inline; po launchu redesignu cala nawigacja zyje w tym pliku. Zmienne uzywane: page.lang — 'pl' lub 'en' (z front-matter strony) page.lang_alt — opcjonalny URL do drugiej wersji jezykowej page.url — Liquid: aktualny URL strony site.data.tags — _data/tags.yml: mapa slugow specjalizacji {%- endcomment -%} {%- assign lang = page.lang | default: 'pl' -%} {%- assign on_spec = page.url | default: '' -%} {%- if lang == 'en' -%} {%- assign spec_active = on_spec | slice: 0, 17 | downcase -%} {%- if spec_active == '/en/specjalizacje/' -%}{%- assign is_spec = true -%}{%- endif -%} {%- else -%} {%- if on_spec contains '/specjalizacje/' -%}{%- assign is_spec = true -%}{%- endif -%} {%- endif -%}
